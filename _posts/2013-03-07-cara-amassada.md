 De manhã, <br/>
 Vai ao trabalho <br/>
 Condução lotada, <br/>

 Vida desorganizada <br/>
 Cara amassada <br/>
 Labuta o dia todo <br/>
 chega dia 5 recebe recompensa <br/>

Moída, <br/>
amassada, <br/>
desfigurada. <br/>

Pela labuta diária. <br/>
Realidade da selva, <br/>
E o proletariado desse Brasil <br/>
Dessa cara amassada.
